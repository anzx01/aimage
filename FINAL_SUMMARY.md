## 快速开始

### 1. 克隆项目
```bash
git clone <repository-url>
cd aimage
```

### 2. 安装依赖
```bash
cd frontend
npm install
```

### 3. 配置环境变量
```bash
cp .env.example .env.local
# 编辑 .env.local 填入 Supabase 配置
```

### 4. 启动开发服务器
```bash
npm run dev
```

### 5. 访问应用
打开浏览器访问: http://localhost:3000
